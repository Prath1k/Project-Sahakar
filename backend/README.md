# Project Sahakar — Autonomous AI Assistant & RAG Engine

This is the backend FastAPI server and Docker container for **Project Sahakar**, deployed on Render.com, providing:
- **SCAAR RAG Engine**: The Reconciliation Engine & Document Ocean for isolated, multi-user memory and document indexing.
- **Dynamic Voice Routing**: Kokoro-based TTS and continuous voice recognition support.
- **Granular Data Control**: Query-based memory wiping (`"wipe my memory"`) and fact forgetting (`"forget..."`).
